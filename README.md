# Data-Driven Health & Fitness Progress Predictor
**Academic Lab Project: Data Interpretation & Predictive Modeling**

## 📌 Project Overview
This MS Excel-based project provides a mathematical framework for interpreting the relationship between caloric deficits and physiological weight change. By utilizing the **Mifflin-St Jeor Equation** for BMR and the **7,700 kcal/kg rule**, the tool forecasts future weight loss trends based on user-inputted nutrition and activity data.

## 🛠️ Technical Features
* **BMR Calculator:** Automated calculation using conditional logic for gender-specific biological variables.
* **Dynamic Tracker:** 30-day logging system for caloric intake and exercise expenditure.
* **Predictive Modeling:** Utilizes **Linear Regression** to generate a weight loss forecast.
* **Statistical Validation:** Includes **R² (Coefficient of Determination)** and **Regression Equations** to verify data reliability.

## 📊 Data Interpretation Analysis
The project focuses on the "Linear Decay" of body mass over time. Key findings from the model include:
- **Slope (m):** Represents the average daily weight loss (kg/day).
- **R² Value:** Demonstrates the strength of the correlation between the energy deficit and mass reduction (Target: >0.95).

## 🚀 How to Use
1. Clone the repository or download the `.xlsx` file.
2. Enter your **Gender, Weight, Height, and Age** in the Input Section (Cells B1:B4).
3. Log your daily **Calories Eaten** and **Exercise Burn** in the Tracker Table.
4. View the **30-Day Forecast Chart** to see your predicted weight-loss trajectory.
